Lab 10 - Semester Project

a. Problem Statement

Task 1: Develop a brief project proposal document for a data-driven web application with the following requirements:

  • State management
  • Form validation
  • Data security
  • User and role-based functionalities

Task 2: Write a complete Software Requirements Specification (SRS) document.

Task 3: Develop analysis and design models including:

  • Use Case Model
  • Sequence Diagrams
  • Activity Diagrams
  • Class Diagrams

Task 3A: Develop Component and Deployment Models.

Task 4: Design and implement database models:

  • ER Model
  • Relational Model
  • Normalization
  • Physical Model
  • SQL Implementation

Task 5: Implement the web application using technologies covered in the course.

Task 6: Host the application on a domain. The database may be hosted locally or online.

Task 7: Perform thorough testing of the web application.
